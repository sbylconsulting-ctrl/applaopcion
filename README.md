# La Opción Producciones

Aplicación web de gestión financiera para espectáculos y funciones teatrales.

Permite registrar ingresos, gastos, acuerdos con salas y consultar resúmenes financieros de forma visual y sencilla.

## Características

- Dashboard con resumen financiero general
- Registro de funciones
- Alta de nuevas funciones e ingresos
- Cálculo automático de:
  - venta bruta
  - venta neta
  - comisiones
  - gastos
  - caché
  - restante para La Opción
- Vista detalle de cada función
- Resumen mensual
- Informes por obra y por sala
- Panel de administración
- Persistencia local con `localStorage`

## Tecnologías

Este proyecto está hecho en:

- HTML
- CSS
- JavaScript nativo

No necesita React, Vite ni instalación de dependencias.
